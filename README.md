# secondary-diagonal-sum-c
A simple C program that prints the secondary diagonal elements of a 3x3 matrix and calculates their sum.

# Secondary Diagonal Sum in Matrix (C Program)

## 📌 Description
This program prints the secondary diagonal elements of a 3×3 matrix and calculates their sum.

## 🚀 How It Works
- Traverse the matrix using a loop.
- Access secondary diagonal elements using arr[i][n-1-i].
- Add each element to the sum.
- Print the elements and the final sum.

## 📥 Input Matrix

```text
1 2 3
4 5 6
7 8 9
```

## 📤 Output

```text
Secondary Diagonal Element = 3
Secondary Diagonal Element = 5
Secondary Diagonal Element = 7

Total Secondary Diagonal Sum = 15
```

## 🛠️ Concepts Used
- 2D Arrays
- Matrix Traversal
- Secondary Diagonal
- Loops
- Sum Calculation

## 🎯 Learning Outcome
- Understanding secondary diagonals
- Using arr[i][n-1-i]
- Working with matrix indexing
- Calculating diagonal sums
